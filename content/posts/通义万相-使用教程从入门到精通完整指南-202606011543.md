## 一、注册与安装：三步开启AI绘画之旅

通义万相是基于网页端的工具，不需要下载任何客户端，这点对电脑配置低的用户非常友好。

### 1. 访问官网
打开浏览器，搜索“通义万相”或直接访问通义千问官网（tongyi.aliyun.com），在顶部导航栏找到“通义万相”入口。

> 📸 **截图位置**：官网首页顶部导航栏，鼠标悬停“产品”菜单，显示通义万相选项

### 2. 注册/登录
- **已有阿里账号**：直接用淘宝、支付宝、钉钉账号扫码登录
- **新用户注册**：点击“免费注册”，填写手机号获取验证码，设置密码即可
- **注意**：目前通义万相支持手机号+验证码登录，无需实名认证（但生成图片有水印，后面会讲如何去除）

### 3. 进入创作界面
登录后自动跳转到控制台，你会看到三个主要模块：
- **文生图**：输入文字描述生成图片
- **图生图**：上传参考图进行二次创作
- **相似图生成**：以图搜图风格变体

> 💡 **小技巧**：建议首次使用时先点击“新手引导”，系统会赠送10个免费积分（每个积分可生成一张图）。

## 二、基础功能使用：3个核心玩法详解

掌握了注册步骤后，我们来逐一拆解**通义万相使用方法**中的三个基础功能。

### 1. 文生图：最核心的功能

这是通义万相最常用的功能，操作极其简单：

**步骤**：
1. 点击“文生图”进入创作页
2. 在输入框填写描述词（Prompts）
3. 选择“风格”（默认“通用”，可选“二次元”“写实”“插画”等）
4. 设置“图片比例”（1:1、4:3、16:9等）
5. 点击“立即生成”，等待5-15秒

**案例**：
- 输入：“一只穿着宇航服的柴犬站在月球表面，背景是地球，风格写实”
- 选择风格：写实
- 比例：1:1
- 生成结果：画面清晰，柴犬毛发细节到位，月球坑洞有质感

> 📸 **截图位置**：文生图界面，输入框已填写描述词，右侧显示生成结果

**关键参数说明**：
| 参数 | 作用 | 建议值 |
|------|------|--------|
| 描述词 | 画面主体+环境+风格+细节 | 20-50字最佳 |
| 负面词 | 排除不想要的元素 | 如“模糊、畸形、多余手指” |
| 图片比例 | 决定构图 | 按用途选择 |

### 2. 图生图：让AI参考你的素材

如果你有一张照片或草图，想让AI基于它生成新图，就用这个功能。

**步骤**：
1. 点击“图生图”
2. 上传参考图片（支持jpg/png，最大10MB）
3. 填写描述词（如“把背景换成樱花林”）
4. 调整“参考强度”（0-1，值越大越贴近原图）
5. 生成

**实用场景**：
- 把真人照片转二次元风格
- 给产品图换背景（如服装模特换场景）
- 修复老照片（配合“去除瑕疵”提示词）

### 3. 相似图生成：快速获取风格变体

这个功能适合设计师找灵感——上传一张图，AI会生成4张风格相似但细节不同的版本。

**步骤**：
1. 点击“相似图生成”
2. 上传参考图
3. 选择“风格相似度”（建议0.6-0.8）
4. 一键生成

> 💡 **进阶用法**：先用文生图生成一张满意的图，再用相似图生成功能批量产出变体，效率翻倍。

## 三、进阶技巧：3个让你效率翻倍的高级玩法

基础功能只能满足日常需求，但如果你想在AI绘画领域玩得更专业，下面这些**通义万相教程**中的进阶技巧值得收藏。

### 技巧1：用“负面词”精准避坑

很多人不知道，通义万相支持**负面词**输入。这是一个极其实用的功能，能帮你避免AI常见的“翻车”情况。

**操作位置**：在文生图输入框下方，点击“高级设置”，找到“负面词”输入框。

**常用负面词清单**：
- 人物类：`畸形、扭曲、额外手指、多余肢体、不对称脸`
- 画质类：`模糊、低分辨率、噪点、水印、文字`
- 风格类：`油画质感、卡通化、现实（看情况使用）`

**案例**：生成“一位穿旗袍的女士”时，加入负面词`多余手指、畸形手`，出图质量明显提升。

### 技巧2：组合Prompt模板化输出

如果你需要批量生成同一主题的图片（比如电商详情页），可以建立自己的Prompt模板。

**模板结构**：
```
[主体描述]，[环境/背景]，[风格]，[色调]，[细节要求]
```

**实战模板**（以“夏日饮品”为例）：
```
主体：一杯冰镇柠檬茶，玻璃杯壁有冷凝水珠
环境：木质桌面上，旁边有薄荷叶和吸管
风格：美食摄影，4K，景深效果
色调：暖黄色调，高饱和度
细节：冰块半透明，柠檬片新鲜多汁
```

你可以将主体替换为“咖啡”“果酒”等，快速产出系列图片。

### 技巧3：利用“图片比例”做排版设计

通义万相支持多种图片比例，但很多人只用了1:1。实际上，不同比例对应不同用途：

| 比例 | 推荐用途 | 提示词调整 |
|------|----------|------------|
| 1:1 | 头像、方形封面 | 主体居中 |
| 4:3 | 横版海报、公众号头图 | 留白区域在左侧 |
| 16:9 | 视频封面、PPT背景 | 重要元素放在右侧1/3 |
| 3:4 | 小红书竖版封面 | 主体靠上，下方留文字区 |
| 9:16 | 手机壁纸、短视频封面 | 纵向构图 |

**技巧**：生成前先在脑中规划好文字排版位置，在Prompt中提示“左侧留白用于文字”，AI会智能安排构图。

## 四、常见问题解答（FAQ）

在使用过程中，用户最常遇到以下问题，我整理了详细解答：

### Q1：生成的图片有水印怎么办？
**答**：通义万相免费版会在图片右下角添加“通义万相”水印。去除方法：
- 方法1：用PS或美图秀秀的“裁剪”功能切掉水印区域（适合构图允许的情况）
- 方法2：在描述词中加入“无水印”，AI会尽量避开（不保证100%）
- 方法3：付费订阅（目前未开放，未来可能推出）

### Q2：为什么生成的图片总是“崩坏”？
**答**：常见原因及解决方案：
- 描述词太短（少于10字）→ 扩充到20-50字
- 包含矛盾描述（如“写实”+“卡通”）→ 统一风格
- 未使用负面词 → 加入“畸形、模糊”等词
- 参考图太复杂 → 简化参考图或降低参考强度

### Q3：通义万相和Midjourney比哪个好？
**答**：各有优劣。通义万相的优势：
- 完全免费（目前）
- 中文理解能力强（MJ需要英文Prompt）
- 操作简单，学习成本低
- 劣势：风格多样性不如MJ，细节表现力稍弱

**建议**：日常快速出图用通义万相，专业级创作用Midjourney。

### Q4：生成的图片可以商用吗？
**答**：根据阿里云服务条款，通义万相生成的图片**可自由使用**，包括商业用途。但需注意：
- 避免生成他人版权角色（如迪士尼人物）
- 不要生成违法内容
- 建议保留生成记录备查

### Q5：每天能免费生成多少张？
**答**：目前每日签到赠送10积分（1积分=1张图），另外完成新手任务可额外获得20-50积分。积分次日刷新。

### Q6：为什么我的图片生成特别慢？
**答**：通常是因为：
- 高峰期服务器负载高（建议避开晚8-10点）
- 描述词过长（超过100字会降低速度）
- 网络环境问题（尝试切换WiFi/4G）

### Q7：如何生成更真实的照片？
**答**：使用以下Prompt技巧：
- 加入相机参数：`f/1.8光圈、ISO 200、35mm镜头`
- 指定光线：`自然光、柔光箱、逆光`
- 添加后期：`经过Lightroom调色、颗粒感`

经过近半年的深度使用，我对**通义万相**的评价是：**AI绘画领域的“傻瓜相机”**——它牺牲了部分专业级控制，换来了极致的易用性。

**推荐人群**：
- 完全零基础的新手（5分钟就能出第一张图）

**我的建议**：
1. **不要过度依赖AI**：AI生成的图片可以作为素材或灵感，但最终作品需要你的创意和后期处理
2. **建立自己的Prompt库**：把好的描述词分类保存，下次直接复用
3. **多尝试不同风格**：通义万相的“二次元”和“插画”风格其实很出色，不要只拘泥于写实
4. **结合其他工具使用**：用通义万相生成底图，再用Photoshop精修，效率和质量都能提升
